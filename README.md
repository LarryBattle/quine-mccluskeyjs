quine-mccluskeyjs
=================

Quine-McCluskeyJS is a Javascript version of the Quine Mccluskey method for Boolean algebra reduction. 
In otherwords, this helps you reduce your expressions and boolean algebra.
Let `*` stand for not.
Change `A* + AB*` into `(AB)*`
